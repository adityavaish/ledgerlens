Ledgerlens

AI assistant inside Excel for finance and accounting teams. Pull data, run analyses, and build reports with natural language — all running locally on your machine.

↓ Install for Windows View on GitHub

Windows 10 or 11 · Excel desktop required · Free & open source

How to install

  1. Download by clicking the "Install for Windows" button above.
  2. Run the downloaded Install-Ledgerlens.cmd file (double-click).
  3. The installer will set up everything — Node.js (if not already installed), the Ledgerlens app, and a shortcut on your Start menu and desktop. Takes about a minute.
  4. Click Ledgerlens on your desktop. Excel will open with the add-in loaded.
  5. Click the Ledgerlens button on the Home tab of Excel to open the chat panel.
About the Windows warning: The first time you run the installer, Windows SmartScreen may show "Windows protected your PC". This appears because the file hasn't been digitally signed yet. To proceed: click More info, then Run anyway. The installer only writes to your own user folder — no admin rights required.

What it does

Natural-language Excel

Generate data, build pivots, format ranges, and write formulas just by describing what you want.

Azure Data Explorer

Query Kusto clusters directly from a worksheet — signs you in with your own account via your browser.

BYO MCP servers

Automatically imports MCP servers you already configured in VS Code, GitHub Copilot CLI, Claude, Cursor, Windsurf, and more.

Runs locally

The server runs on your laptop. Your data and queries never go to a shared cloud host.

Auto-updates

Every launch quietly checks for a new release on GitHub and upgrades in place. Skip with one env var if you'd rather control updates.

Open source

MIT-licensed, every line on GitHub. Fork it, audit it, or send a pull request.

Updating & uninstalling

Update: Ledgerlens checks GitHub Releases for a newer version every time you launch it; no action needed.

Uninstall: Delete the %LOCALAPPDATA%\ledgerlens and %LOCALAPPDATA%\Programs\ledgerlens folders, and remove the Start-menu and desktop shortcuts. That's everything.
